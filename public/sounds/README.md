# Page Turn Sound

This directory should contain `page-turn.mp3`.

## How to add the sound

1. Download a free CC0 paper rustle / page turn sound from freesound.org.
   Recommended: https://freesound.org/people/ricardus/sounds/18676/
   or search: https://freesound.org/search/?q=page+turn&filter=license%3A%22Creative+Commons+0%22

2. Convert / rename the file to `page-turn.mp3`

3. Place it in this directory: `/public/sounds/page-turn.mp3`

The audio is loaded with a standard HTML Audio element at 35% volume.
If the file is missing, page turns will be silent (no errors thrown).
